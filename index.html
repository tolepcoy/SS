Tolep server_

Setiap 1 bit data transfer dari jaringan server ini ane enkripsi 3 lapis.
Samo setiap chat yang di store ke database server ane, ane enkripsi jugo :
- Ane enkripsi di lokal storage pake Javascript
- Pas nak ngirim ane enkripsi lagi ke CryptoAES 256bit
- Pas dikirim ane enkripsi lagi jadi Base64
- Pas nyampe di database ane enkripsi manual pake ~REGEX
- Pas di transfer ke chatbox dengan onSnapshot baru ane dekripsi galo
- Sistem Upload & Download data transfer samo cak itu galo
- Setiap 24 jam dari ane login, otomatis chat dihapus!

Jadi chat disini dak pacak di sadap, di embek,
katek riwayat, katek jejak, katek notif, katek bunyi, katek galo-galo.

Jadi katek embekan nian memang aplikasi ini ni, screenshot be ku disable.
Yang pacak nyingok data di server ini kuraso cuman wong gilo samo wong sok tau bae.

Mokaseh.
